# 🌳 府中ものしりクイズ

東京都府中市の歴史・名所・グルメを楽しく学べる、ケヤキの妖精「けやぴ」と一緒に挑戦するミニクイズアプリです。

![けやぴ](./assets/keyapi-wave.webp)

## ✨ 特徴

- 🎴 **全100問** ／ 歴史（30問）・名所（35問）・グルメ文化（35問）の3カテゴリー
- 🗺️ **お出かけマップ** ／ 府中市内の25スポットを巡ってスタンプを集めよう
- 📚 **出典明示** ／ 各問題に信頼できる一次情報源を表示
- 🌱 **3段階の難易度**（初級・中級・上級）
- 🌳 **オリジナルキャラクター「けやぴ」** が問題ごとにコメント
- 📱 **スマホ最適化** ／ PWA対応（ホーム画面に追加可能）
- 🌐 **オフライン動作** ／ 一度開けばネットなしで遊べる
- 🎨 **和風デザイン** ／ 大國魂神社のイメージカラー

## 🛠️ GitHub Pages での公開手順

1. このフォルダの中身をGitHubリポジトリにプッシュ
2. リポジトリの **Settings** → **Pages** を開く
3. **Source** で `main` ブランチ（または `gh-pages`）を選択
4. 数分後、`https://<ユーザー名>.github.io/<リポジトリ名>/` でアクセス可能に
